总结

  1. 普通函数，参数输入的顺序是固定的，同时参数的个数输入要求也是一致的。(如果不一致，需要指定参数名称和值，如func(b=100,a=200))
  2. 带默认值函数，具备默认值的参数可以不输入值，但顺序需要一致
  3. 带*号的参数，可以接受常规参数后剩下的所有参数，该参数为一个tuple
  4. 带**号参数，可以接受常规参数后的参数，但在调用的时候，需要指定参数的名称和值，该参数以dict存储的。
